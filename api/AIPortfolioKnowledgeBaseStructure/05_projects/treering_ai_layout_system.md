\f0\fs24 \cf0 # TreeRing AI Layout System\
\
## Project Overview\
The TreeRing AI Layout System introduced an AI-powered design assistant that automatically generates yearbook page layouts based on uploaded photos and metadata.\
\
## Industry\
EdTech / Consumer Software\
\
## Organization\
TreeRing\
\
## My Role\
Lead Product Designer\
\
## Problem\
Users struggled with blank canvas paralysis when designing yearbook pages.\
\
## Design Strategy\
Introduce AI-assisted layouts that generate multiple page options while allowing users to customize the results.\
\
## Key UX Decisions\
AI layout generation  \
Editable automated suggestions  \
Guardrails for print quality\
\
## Systems Thinking\
The system integrated AI generation, photo metadata analysis, and print constraints.\
\
## Impact\
Reduced time required to design pages}

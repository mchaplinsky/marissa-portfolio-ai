\f0\fs24 \cf0 # CAMP Courthouse Wayfinding System\
\
## Project Overview\
The CAMP project examined wayfinding challenges across courthouses in British Columbia and designed a scalable signage system to improve navigation for visitors and staff.\
\
## Industry\
Government / Public Infrastructure\
\
## Organization\
BC Ministry of the Attorney General\
\
## My Role\
Senior UX Designer\
\
## Project Scope\
Research and design of a universal wayfinding system across 89 courthouses.\
\
## Problem\
Visitors struggled to navigate courthouse environments, requiring staff to constantly assist with directions.\
\
## Users\
Public visitors  \
Court staff  \
Security personnel\
\
## Research Methods\
Site visits  \
Wayfinding audits  \
User interviews  \
Best-practice research\
\
## Design Strategy\
Develop a universal wayfinding framework adaptable to different courthouse sizes.\
\
## Deliverables\
Wayfinding guidelines  \
Prototype signage system  \
User testing\
\
## Tools Used\
Mural  \
Research frameworks\
\
## Impact\
Improved visitor navigation  \
Reduced staff workload}

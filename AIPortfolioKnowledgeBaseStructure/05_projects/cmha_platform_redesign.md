\f0\fs24 \cf0 # CMHA Mental Health Platform Redesign\
\
## Project Overview\
The Canadian Mental Health Association (CMHA) project focused on identifying usability issues across several CMHA digital platforms and redesigning the information architecture to improve accessibility and user engagement.\
\
## Industry\
Mental Health / Healthcare\
\
## Organization\
Canadian Mental Health Association\
\
## My Role\
Senior UX Designer\
\
## Project Scope\
Conduct a comprehensive UX audit of multiple CMHA websites and platforms to identify barriers preventing users from signing up for programs and accessing resources.\
\
## Problem\
CMHA experienced declining program sign-ups and suspected usability issues within their digital ecosystem.\
\
## Users\
Mental health program participants  \
CMHA staff  \
Public users seeking mental health resources\
\
## Research Methods\
Website audits  \
Google Analytics analysis  \
User interviews  \
Accessibility reviews\
\
## Design Strategy\
Improve information architecture and navigation so users could more easily find relevant mental health resources.\
\
## Key UX Decisions\
Future-state site map redesign  \
Improved navigation structure  \
Accessibility improvements\
\
## Systems Thinking\
The project involved analyzing several interconnected CMHA websites and identifying structural issues across the ecosystem.\
\
## Deliverables\
Site audit documentation  \
Future-state information architecture  \
Design recommendations\
\
## Tools Used\
Google Analytics  \
Figma  \
Accessibility testing tools\
\
## Impact\
Improved discoverability of mental health programs  \
Provided roadmap for future platform redesign\
\
## Key Skills Demonstrated\
Healthcare UX  \
Information architecture redesign  \
Research-driven product strategy}

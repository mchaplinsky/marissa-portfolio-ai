# TreeRing AI Layout System

## Project Overview
The TreeRing AI Layout System introduced an AI-powered design assistant that automatically generates yearbook page layouts based on uploaded photos and metadata. The system analyzes user content and produces multiple layout suggestions that users can customize and refine.

The goal was to help users overcome the difficulty of starting from a blank page while preserving creative control over the final design.

## Industry
EdTech / Consumer Software

## Organization
TreeRing

## My Role
Lead Product Designer

## Project Scope
Design an AI-assisted layout system integrated directly into the yearbook creation editor. The system needed to automatically generate page layouts using user-uploaded photos while respecting print constraints and layout best practices.

The system also needed to support user customization and allow users to edit AI-generated results.

## Problem
Many users struggled with "blank canvas paralysis" when designing yearbook pages. Users often had difficulty deciding where to place photos, how many photos to include, and how to structure pages visually.

This resulted in several challenges:

- slow page creation workflows  
- inconsistent layout quality  
- user frustration when designing pages manually  
- increased support requests from users unsure how to start  

## Users
- Teachers managing school yearbooks  
- Parent volunteers designing pages  
- Students contributing content  

Most users had little to no design experience, making intuitive layout tools critical.

## Research Methods
- User interviews with yearbook creators  
- Support ticket analysis  
- Product usage analysis  
- Usability testing of layout workflows  

## Design Strategy
Introduce AI-assisted layouts that automatically generate multiple page options based on the photos users upload.

Rather than replacing design decisions entirely, the system provides layout suggestions that users can review, modify, and customize.

The system focuses on **human-AI collaboration**, where AI accelerates layout creation while users maintain creative control.

## Key UX Decisions

### AI Layout Generation
The system analyzes uploaded photos and generates complete page layouts using automated composition rules.

Layouts consider factors such as:

- photo orientation  
- number of photos available  
- visual balance across the page  

### Editable Automated Suggestions
Users can modify any AI-generated layout.

Users can:

- move photos  
- resize elements  
- swap images  
- delete layout elements  

This ensures that the system accelerates design while preserving flexibility.

### Guardrails for Print Quality
Yearbooks must meet strict print production requirements.

The system includes guardrails that prevent common print issues such as:

- low resolution photos  
- images placed within the binding gutter  
- layouts that exceed print margins  

These guardrails help ensure pages remain production-ready.

## Systems Thinking
The AI layout system required coordination across several interconnected components:

- photo metadata analysis  
- layout generation logic  
- print production constraints  
- user editing tools  

The design required balancing automation with control so that the AI system could accelerate layout creation without limiting user creativity.

\f0\fs24 \cf0 # AI Trust, Transparency, and Control\
\
Trust is one of the most important factors in the success of AI-powered products.\
\
Users must feel confident that the system behaves reliably and that they maintain control over outcomes.\
\
Marissa focuses on designing AI experiences that prioritize trust, transparency, and user control.\
\
---\
\
## Transparency\
\
Transparency helps users understand how the system operates.\
\
Examples include:\
\
\'95 explaining AI suggestions  \
\'95 showing system reasoning when possible  \
\'95 clearly labeling automated outputs\
\
Transparency reduces confusion and builds trust.\
\
---\
\
## User Control\
\
AI systems should allow users to modify or override system suggestions.\
\
Control mechanisms may include:\
\
\'95 editing AI outputs  \
\'95 selecting between multiple options  \
\'95 disabling automated suggestions\
\
These interactions help users maintain agency.\
\
---\
\
## Building Trust Over Time\
\
Trust develops gradually as users observe consistent system behavior.\
\
Designing reliable and predictable AI interactions helps users become comfortable relying on AI assistance.\

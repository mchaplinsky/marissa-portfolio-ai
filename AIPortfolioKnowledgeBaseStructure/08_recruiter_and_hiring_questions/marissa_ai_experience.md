## AI Layout Generation System

One example of Marissa's AI design work is the TreeRing AI Layout System.

This AI-powered feature generates yearbook page layouts based on uploaded photos and image metadata. The system analyzes visual content and composition patterns to produce multiple layout suggestions that users can refine and customize.

Key design considerations included:

• designing AI-generated layout suggestions rather than fixed outputs  
• enabling users to edit and override automated results  
• implementing guardrails to prevent print-quality issues  
• designing system messaging that explains AI decisions  

This approach allowed the system to reduce blank-canvas friction while preserving user creativity and control.
